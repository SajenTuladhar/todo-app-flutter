# To do App 📝

A Flutter To Do application built using the Provider state management pattern, featuring dynamic task updates, clean UI architecture, and responsive task handling.

---

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Real-time UI updates using Provider
- Clean and responsive Flutter UI
- Stateful data management

---

## Tech Stack

- Flutter
- Dart
- Provider
- Material Design

## State Management

This project uses the **Provider** package for efficient and scalable state management.

Implemented concepts include:

- ChangeNotifier
- Consumer
- Provider tree structure
- Centralized app state
- Reactive UI rebuilding

## How to Run

1. Clone the repository

```bash
git clone https://github.com/SajenTuladhar/todo-app-flutter.git
```

2. Navigate into the project

```bash
cd todo-app-flutter
```

3. Install dependencies

```bash
flutter pub get
```

4. Run the application

```bash
flutter run
```
